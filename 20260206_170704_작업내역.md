## 해결된 이슈

### 1. Vercel 빌드 실패 (Cannot find module)
- **원인**: git add 시 특정 파일만 추가하여 많은 파일 누락
- **해결**: `git add -A`로 모든 파일 추가 후 재커밋

### 2. Git Push 인증 에러
- **원인**: SSH/HTTPS 인증 문제
- **해결**: 전체 URL 사용 `git push https://github.com/infoamous-source/kiosk.git main`

### 3. 브라우저 캐시로 이전 버전 표시
- **원인**: 브라우저가 이전 "Korea Starter Pack" 페이지 캐시
- **해결**: Ctrl + Shift + R (강력 새로고침)
