## 1. 제안 배경 및 필요성

### 1-1. 현황

국내 체류 외국인은 **250만 명을 초과**(법무부, 2024)하였으며, 결혼이민자·외국인 근로자·유학생 등 장기 체류 이주민이 지속적으로 증가하고 있습니다. 이들이 한국 사회에 안정적으로 정착하기 위해서는 **디지털 생활 인프라에 대한 이해와 활용 능력**이 필수적입니다.

### 1-2. 이주민이 겪는 핵심 어려움

| 영역 | 구체적 어려움 |
|------|-------------|
| **언어 장벽** | 한국어 메뉴·안내문 해독 불가, 의사소통 단절 |
| **교통 이용** | 택시 호출·지하철 환승 등 대중교통 이용의 어려움 |
| **정보 비대칭** | 구글맵 한국 내 제한, 한국형 앱 생태계 미인지 |
| **생활 서비스** | 배달·쇼핑·중고거래 등 일상 서비스 접근 불가 |
| **안전** | 재난 문자 해독 불가, 긴급 상황 대처 미숙 |

### 1-3. 교육의 필요성

한국은 **세계 최고 수준의 디지털 인프라**를 갖추고 있으나, 이주민들은 이를 충분히 활용하지 못하고 있습니다. 본 교육은 이주민들이 스마트폰을 활용하여 한국 생활의 핵심 서비스에 즉시 접근할 수 있도록 지원하여, **자립적 생활 능력을 향상**시키고 **사회 통합을 촉진**하는 것을 목표로 합니다.

## 2. 교육 목표

### 2-1. 최종 목표
> 이주민이 스마트폰 앱을 활용하여 한국 생활의 핵심 영역(이동, 소통, 식사, 쇼핑, 안전)을 **스스로 해결할 수 있는 디지털 자립 역량**을 갖추도록 한다.

### 2-2. 세부 목표

| 목표 | 기대 성과 |
|------|----------|
| **디지털 생활 인프라 이해** | 한국형 앱 생태계(카카오·네이버 등)의 구조를 이해한다 |
| **필수 앱 설치 및 활용** | 7개 생활 영역의 14개 필수 앱을 직접 설치하고 기본 기능을 사용할 수 있다 |
| **자율 학습 도구 확보** | 교육용 웹앱(Korea Starter Pack)을 통해 교육 후에도 스스로 앱을 찾고 설치할 수 있다 |
| **안전 대응 능력 확보** | 재난 상황 시 안전디딤돌 앱을 통해 정보를 확인하고 대처할 수 있다 |

## 4. 교육 방법론

### 4-1. 교육 방식

```
[설명 10분] → [시연 10분] → [실습 20분] → [확인 10분]
     ↓              ↓              ↓              ↓
  핵심 개념       강사 시연     수강생 직접     질의응답
  간결 전달      화면 공유      앱 조작       문제 해결
```

### 4-2. 교육 특징

| 특징 | 설명 |
|------|------|
| **실습 중심** | 전체 교육시간의 60% 이상을 수강생 직접 실습으로 운영 |
| **다국어 보조** | 교육 웹앱(Korea Starter Pack)이 영어·중국어·일본어를 지원하여 언어별 보조자료 역할 |
| **즉시 활용** | 교육 중 설치한 앱을 교육 종료 후 바로 실생활에 활용 가능 |
| **자율학습 연계** | 웹앱을 홈화면에 설치하여 교육 후에도 새로운 앱을 자율적으로 탐색·설치 가능 |
| **OS 자동 감지** | iOS/Android 기기를 자동 감지하여 해당 앱스토어로 바로 연결 |

### 4-3. 교육 도구

| 도구 | 용도 |
|------|------|
| **Korea Starter Pack 웹앱** | 앱 안내·설치 가이드 (PWA, 다국어 지원) |
| **빔 프로젝터 / 대형 모니터** | 강사 시연 화면 공유 |
| **수강생 개인 스마트폰** | 실습용 기기 (개인 기기 활용) |
| **다국어 실습 워크시트** | 단계별 실습 가이드 (인쇄물) |

## 8. 교육 플랫폼 소개: Korea Starter Pack

### 8-1. 개요

| 항목 | 내용 |
|------|------|
| **접속 주소** | https://infoamous-source.github.io/kiosk/ |
| **형태** | PWA (Progressive Web App) – 설치 없이 웹 접속, 홈화면 추가 가능 |
| **지원 언어** | 한국어, 영어, 중국어, 일본어 |
| **지원 기기** | iOS / Android 자동 감지 |
| **비용** | 무료 |

### 8-2. 주요 기능

- ✅ **카테고리별 앱 탐색**: 7개 생활 카테고리로 분류된 14개 필수 앱
- ✅ **원클릭 설치**: 앱 카드 터치 시 해당 OS의 앱스토어로 자동 이동
- ✅ **다국어 UI**: 수강생 모국어로 앱 설명 확인 가능
- ✅ **신뢰도 뱃지**: 정부추천 / 현지인필수 / 외국인친화 뱃지로 앱 신뢰도 시각화
- ✅ **PWA 지원**: 홈화면에 설치하여 앱처럼 상시 접근 가능
- ✅ **오프라인 접근**: 설치 후 인터넷 없이도 앱 정보 확인 가능
